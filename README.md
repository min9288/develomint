## 서비스 개요      
기존 틀에 박힌 개발자 커뮤니티에 탈피하여, 개발 프로젝트 팀 모집부터 구인구직, 공모전, 프리랜서 서비스 등 개발자를 위한 다양한 서비스들을 제공하는 종합 개발 커뮤니티 플랫폼입니다.
